Mittelwert¶

Mit dem Begriff Mittelwert (auch Durchschnitt genannt) ist in der Statistik meistens das artithmetische Mittel gemeint. Der Mittelwert ist dabei eine Kennzahl fr die zentrale Tendenz einer Verteilung. Das arithmetische Mittel ist rechnerisch die Summe der gegebenen Werte geteilt durch die Anzahl der Werte. Angewandt kann dies beispielsweise zum Berechnen einer Durchschnittsgeschwindigkeit werden. In diesem Fall wrden die Werte als Geschwindigkeit interpretiert werden. Der Erwartungswert einer Zufallsvariablen beschreibt hingegen die Zahl, die die Zufallsvariable im Mittel annimmt.

Standardabweichung¶

Die Standardabweichung ist ein Ma dafr, wie weit einzelne Datenstze verteilt sind. Mathematisch wird die Standardabweichung definiert als die mittlere quadratische Abweichung einer reellen Zufallsvariablen von ihrem Erwartungswert. Wenn alle Werte gleich sind, so ist die Standardabweichung Null, da diese von der Varianz abgeleitet ist. Das heit je weiter Daten vertreut sind, desto hher ist die Standardabweichung. Je enger/nher Datenstze jedoch beieinander liegen, desto niedriger fllt die Standardabweichung aus. Es gibt zwei verschiedene Formeln der Anwendung, welche sich darin unterscheiden, dass nicht durch n, sondern durch n-1 geteilt wird.

def moments(x):
    mean = x.mean()
    std = x.var()**0.5
    skewness = ((x - mean)**3).mean() / std**3
    kurtosis = ((x - mean)**4).mean() / std**4
    return [mean, std, skewness, kurtosis]

def fftfeatures(wavdata):
    f = numpy.fft.fft(wavdata)
    f = f[2:int(f.size / 2 + 1)]
    f = abs(f)
    total_power = f.sum()
    f = numpy.array_split(f, 10)
    return [e.sum() / total_power for e in f]

def read_wav(wav_file):
    """Returns two chunks of sound data from wave file."""
    w = wave.open(wav_file)
    n = 50 * 10000
    if w.getnframes() < n * 3:
        raise ValueError('Wave file too short')
    #For each music file 2 sequences, each containing n frames are subtracted. The first sequence starts at postion n,
    #the second sequence starts at postion 2n. The reason for extracting 2 subsequences is, that later on we like to
    #find the best features and in this exercise we assume that good features have the property that they are similar for 2 subsequences
    #of the same song, but differ for subsequences of different songs.
    w.setpos(n)
    frames = w.readframes(n)
    wav_data1 = struct.unpack('%dh' % n, frames)
    frames = w.readframes(n)
    wav_data2 = struct.unpack('%dh' % n, frames)
    return wav_data1, wav_data2

# euklidische Distanz zu allen Teilsequenzen
def calcEuclideandDist(df_one, df_two):
    euclid_dist_dict = {}
    for index_one, row_one in df_one.iterrows():
        euclid_dist_list = []
        for index_two, row_two in df_two.iterrows():
            euclid_dist_list.append([distance.euclidean(row_one, row_two), index_two])
        euclid_dist_list.sort()
        euclid_dist_dict[index_one] = euclid_dist_list
    return euclid_dist_dict

def calcCorrelationDist(df_one, df_two):
    cor_dist_dict={}

Der Genetische Algorithmus¶

Ein Genetischer Algorithmus (GA) ist eine Such-Heuristik, welche durch Charls Darwin's Theorie der natrlichen Evolution inspiriert wurde. Dieser reflektiert den Prozess, in welchem eine natrliche Selektion der besten (fittest) Individuen einer Population fr eine Reproduktion selektiert werden. Genetische Algorithmen (GA) sind sehr gut fr Probleme in der Suche, als auch fr Optimierungen einzusetzen. Ein Beispiel hierfr ist der Einsatz eines GA, um eine Lsung fr das "Travelling Salesman Problem" (TSP) zu finden. Fr die Erkundung der besten Merkmale in diesem Fall werden die einzelnen Elemente des GA auf die Problemstellung wie folgt bertragen:

  • Gene: Einzelnes Element eines Merkmals, bzw. ein Satz an Parametern (Variablen).
  • Individual/Chromosome: Ein Chromosom ist eine Zusammensetzung von Genen. In diesem Fall ein einzelnes Merkmal, welches die Bedingungen erfllt.
  • Population: Eine Sammlung mglicher Merkmale.
  • Parents: Zwei Merkmale, welche kombiniert werden, um ein neues Merkmal zu generieren.
  • Mating Pool: Eine Sammlung an Elternteilen, welche dazu verwendet werden, eine neue Population (nchste Generation) zu generieren.
  • Fitness: Eine Funktion, welche die Gte der Mermale anhand ihres mittleren Rangs bewertet.
  • Mutation: Eine Mglichkeit Variationen in der Population zu erzeugen, durch zuflliges austauschen von Elementen der Merkmale.
  • Elitism: Eine Mglichkeit die besten Individuen in die nchste Generation zu bertragen.

Der hier dargestellte Genetische Algorithmus (GA) wird die folgenden Schritte ausfhren:

  1. Erzeugung einer initialen, zuflligen Population.
  2. Fitness der Individuen (Chromosomen) innerhalb der Population berechnen.
  3. Selektion des Mating Pools, d.h. der fittesten Individuen.
  4. Kreuzung zur Erzeugung einer neuen Generation.
  5. Mutation.
  6. Austausch gegen die neue Population.
  7. Wiederhole von Schritt 1 bis 6, bis die Abbruchbedingung erfllt ist.

def crossover(parent1, parent2):
    child = []
    childP1 = []
    childP2 = []
    # Selection of random genes
    geneA = int(random.random() * len(parent1))
    geneB = int(random.random() * len(parent1))
    
    startGene = min(geneA, geneB)
    endGene = max(geneA, geneB)
    
    for i in range(startGene, endGene):
        childP1.append(parent1[i])
    
    childP2 = [item for item in parent2 if item not in childP1]
    # concatenate both
    child = childP1 + childP2
    return child

def crossoverPopulation(matingpool, eliteSize):
    children = []
    length = len(matingpool) - eliteSize
    pool = random.sample(matingpool, len(matingpool))
    for i in range(0, eliteSize):
        children.append(matingpool[i])
    
    for i in range(0, length):
        child = crossover(pool[i], pool[len(matingpool)-i-1])
        children.append(child)
    return children
Mutationsfunktion¶

Es gibt verschiedene Varianten, um Mutation zu erzeugen. Diese sind unter Anderem: it flip, swap, inverse, uniform, non-uniform, gaussian, shrink. Mutation hlt eine wichtige Funktion fr GAs inne, denn diese hilft lokale Komvergenz (local convergence), durch das Einfhren neuer, noch unbekannter Merkmale, zu vermeiden. Die Einfhrung neuer Merkmale ermglicht es einen noch unbekannten Lsungsraum zu erkunden. Da einzelne Merkmale nicht einfach herausgelscht werden drfen, wird hier die Methode des swap mutation angewandt. Dies bedeutet, dass mit einer geringen Wahrscheinlichkeit verschiedene Merkmale ihre Lokation austauschen (swap) werden. Fr ein Individuum kann dies mittels der folgenden Funktion erzeugt werden.

def mutation(individual, mutationRate):
    for swapped in range(len(individual)):
        if(random.random() < mutationRate):
            swapWith = int(random.random() * len(individual))
            
            feature1 = individual[swapped]
            feature2 = individual[swapWith]
            # swap positions
            individual[swapped] = feature2
            individual[swapWith] = feature1
    return individual
